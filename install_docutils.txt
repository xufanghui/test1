安装docutils
1.下载docutils

wget --no-check-certificate https://pypi.python.org/packages/source/d/docutils/docutils-0.12.tar.gz#md5=4622263b62c5c771c03502afa3157768


2.安装docutils
tar -xzvf docutils-0.12.tar.gz
cd docutils-0.12
python setup.py install
